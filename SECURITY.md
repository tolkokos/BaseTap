# Security Policy

## Supported Versions

We actively support the latest release and pre-releases on Base networks.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please email security@basetap.xyz or create a private security advisory on GitHub.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and work with you to resolve the issue before public disclosure.

## Scope

This security policy applies to:
- BaseTap smart contract (BaseTap.sol)
- Deployment and upgrade scripts
- GitHub Actions workflows

Out of scope:
- Third-party dependencies (report to their maintainers)
- UI/frontend components (separate security process)
